# 五金GOGO購 品牌官網專案

## 專案背景
台中在地五金工具商，品牌名稱「五金GOGO購」，自民國69年（1980年）起在地經營，超過45年歷史。
主要銷售日本、歐洲專業手工具，代理超過30個品牌。
目前已有 ERP 系統（HOLI_ERP.html）管理銷售數據。

## 專案目標
製作「品牌網站」（非購物網站），展示品牌形象與代理工具。
訪客在網站上瀏覽品牌與商品資訊，點擊商品才導向 iOPEN 購買。
主要導流目標為 iOPEN 線上賣場（無手續費），減少對蝦皮（手續費14.5%）的依賴。
網站上不做購物車、價格比較等電商功能，如需電商直接用 iOPEN。

## 網站需求
### 頁面結構（以 test.html 為準）
1. **首頁 Hero**：全幅背景圖＋品牌標語，預計改為輪播（使用品牌形象圖 新-2~新-7）
2. **代理品牌**：左側品牌選單 + 右側內容面板（預設顯示品牌總覽，點擊品牌顯示商品）
3. **關於我們**：左文右圖（圖用 hand tools.JPG），下方三個品牌價值（專業、熱忱、信任）
4. **聯絡我們**：電話、LINE、賣場連結、地圖照片（尚未製作）

### 關於我們文字（已確認）
- 第1段：從民國69年起，我們專注在手工具這個領域已經超過45年。
- 第2段：目前代理日本、德國、美國超過30個專業工具品牌，從鉗子、六角板手到量測儀器，每個品牌都是長期配合、品質確認過的。
- 第3段：品牌舉例（KEIBA 鉗子 — 專業技師指定、BONDHUS 六角板手 — 科技業設備維護、WERA 起子組 — 人體工學設計）
- 第4段：協助客人選工具，歡迎聯繫

### 品牌價值（已確認）
- **專業**：45年銷售經驗，協助挑選最適合的工具
- **熱忱**：專業技師或DIY新手都歡迎詢問
- **信任**：全部正品公司貨，快速出貨、貼心服務
